Dyna dataset. 

FILES: 
- dyna_{male,female).h5: dataset files. 
- dyna_female_small.h5: sample dataset with only one sequence for fast download
- training_file_{male,female}.txt: file containing subject ids and sequences

PYTHON:
load_dyna_dataset.py: Loads a queried sequence
display_dyna_dataset.py: Displays a queried sequence

MATLAB:
load_dyna_dataset.m: Loads a queried sequence
visualize_mesh_sequence.m: displays a sequence of meshes
